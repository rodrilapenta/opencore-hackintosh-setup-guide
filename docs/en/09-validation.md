# 09 - Validation checklist

After post-install, run:

```bash
./scripts/01-verify-post-install.sh
```

Then manually validate:

- Internal display.
- External display.
- Graphics acceleration / Metal.
- Internal speakers.
- Headphone jack.
- Internal microphone.
- USB2.
- USB3.
- Webcam.
- Bluetooth.
- Wi-Fi.
- Ethernet.
- Battery status.
- Brightness keys.
- Sleep/wake.
- Card reader.
- Optical drive, if present.
- iPhone USB / Finder detection.
- Apple services only after SMBIOS sanity check.
