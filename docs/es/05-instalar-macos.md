# 05 - Bootear e instalar macOS

## Pasos

1. Bootear desde el pendrive.
2. Elegir OpenCore.
3. Elegir el instalador de macOS.
4. Formatear el disco destino si hace falta.
5. Iniciar la instalación.
6. Dejar que reinicie todas las veces necesarias.
7. En cada reinicio, volver a bootear usando OpenCore.
8. Continuar hasta llegar al asistente inicial de macOS.

## Notas

El instalador puede mostrar tiempos muy inexactos.

Puede parecer trabado durante un rato. Tené paciencia salvo que haya kernel panic claro o loop de reinicios.

## Después del primer arranque

Cuando macOS bootea:

1. Saltá Apple ID por ahora si todavía no validaste SMBIOS.
2. Llegá al Escritorio.
3. Seguí con la instalación de OpenCore en el disco interno.
