# 09 - Checklist de validación

Después de la post-instalación, corré:

```bash
./scripts/01-verify-post-install.sh
```

Después validá manualmente:

- Pantalla interna.
- Pantalla externa.
- Aceleración gráfica / Metal.
- Parlantes internos.
- Jack de auriculares.
- Micrófono interno.
- USB2.
- USB3.
- Webcam.
- Bluetooth.
- Wi-Fi.
- Ethernet.
- Estado de batería.
- Teclas de brillo.
- Reposo/despertar.
- Lector de tarjetas.
- Lector óptico, si existe.
- iPhone por USB / Finder.
- Servicios Apple recién después de validar SMBIOS.
